# 🎓 Portal by Raghu - The Infinite Learning Engine

A premium, state-of-the-art educational platform designed to provide an endless supply of high-quality practice material for students. Featuring a procedural generation engine, global API integration, and a stunning modern UI, **Portal by Raghu** is built to make learning accessible, infinite, and engaging.

---

## 🚀 Key Features

### ♾️ Infinite Content Engine
Unlike traditional study apps limited by static databases, this portal features a custom-built **Procedural Generation Logic**:
- **Maths**: Unlimited variations of questions on Percentages, Profit/Loss, Ratios, and Arithmetic.
- **Reasoning**: Infinite Number and Alphabet series puzzles generated on-the-fly.
- **Dynamic Explanations**: Every generated question comes with a step-by-step, logical explanation to ensure concept mastery.

### 🌍 Multi-language Support (Hindi / English)
Full inclusivity for learners across India:
- **Instant Toggle**: Switch the entire interface between Hindi and English with a single click.
- **Translated Content**: Not just the UI, but the generated questions and explanations adapt to the selected language.

### 📅 Global Daily Quiz (Synchronized)
A fresh 50-question challenge delivered to every user, every single day:
- **Seeded Stability**: Uses date-based seeds and local caching to ensure the quiz remains identical for the entire day across all sessions.
- **Hybrid Mix**: 50% Procedural Logic, 30% Global Trivia API, and 20%Sequentially Rotated Premium Pool.
- **Performance Tracking**: Detailed results with negative marking, time tracking, and a comprehensive answer key with explanations.

### 💎 Premium Design & UX
A visually stunning interface built for focus and delight:
- **Interactive UI**: Powered by `Framer Motion` for smooth, glassmorphic transitions and micro-animations.
- **Mesh Gradients**: A modern, vibrant aesthetic that feels professional and premium.
- **Responsive Navigation**: Optimized for both Desktop and Mobile (PWA ready).
- **Lucide Icons**: Clean, intuitive iconography for better navigation.

### 🧠 Smart Learning System
- **Zero-Repeat Logic**: A persistent history tracker ensures you don't see the same static questions twice until the entire pool is exhausted.
- **Learning Mode**: Focused practice sessions with categorized subjects (GK, Reasoning, Maths, English).
- **Auto-Update**: Built-in logic to rotate subjects and refresh content goals.
